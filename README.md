# RoomFlow
스터디룸 예약 관리 시스템 (학습 목적 프로젝트)
## 프로젝트 개요
스터디카페/워크스페이스의 스터디룸을 온라인으로 예약하고 관리하는 서비스입니다
Spring MVC 패턴과 비즈니스 로직 설계를 학습하기 위해 제작했습니다.
## 타겟 사용자
- **관리자**: 스터디룸 등록/관리
- **일반 사용자**: 스터디룸 예약/취소
## 구현 기능
### 사용자 인증
- [x] 회원가입
- [x] 로그인 (Cookie 기반)
- [x] 로그아웃
- [x] 사용자 Role 구분 (USER/ADMIN)
- [ ] 비밀번호 암호화
- [x] Session 기반 인증

### 스터디룸 관리 (관리자)
- [x] Room 도메인 설계
- [ ] 스터디룸 등록
- [ ] 스터디룸 목록 조회
- [ ] 스터디룸 수정
- [ ] 스터디룸 삭제

### 예약 관리 (사용자)
- [x] Reservation 도메인 설계
- [x] 예약 생성
- [ ] 예약 목록 조회
- [ ] 예약 취소
- [ ] 중복 예약 방지 로직
- [ ] 예약 취소 시간 제한 (30분 전)
- [ ] 동시 예약 수 제한
### 기타
- [ ] 권환확인 구현
- [ ] 도메인별 서비스 분리 
- [ ] DTO 분리
- [ ] 예외 처리 체계화
## 학습 목표
1. Spring MVC 패턴의 계층별 책임 분리 실습
2. 예약 시스템의 중복 방지 로직 구현
3. 비즈니스 요구사항을 코드로 구조화하는 과정 학습
4. 강의를 먼저 듣고 따라하는 방식이 아닌, 구현 중 문제를 마주했을 때 필요한 개념을 찾아 학습하는 동기 기반 학습 방식 실천
## 기술 스택
- Java 17
- Spring Boot 4.0.3
- Thymeleaf
